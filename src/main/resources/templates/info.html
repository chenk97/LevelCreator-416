TOOLS

Saves the current project.

Shows a preview of the map currently being worked on.

Export maps as .png.

Provides a quick way of filling empty areas or areas covered with the same tiles.

Download allows users to download the .JSON file of current map that they are working on.

Allows users to select tiles on the map.

Allows users to erase selected tiles on the map.

Allow users to draw there own tileset on a canvas.

Allows user to add collaborators to their project.

KEYBOARD SHORTCUTS

Save: Ctrl+S

Erase: Delete

Copy: Ctrl+C

Paste: Ctrl+V

Drag Map: Alt+Mouse Drag

Zoom In and Out: Ctrl+Mouse Wheel Up and Down
